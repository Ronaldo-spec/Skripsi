# Report Security Note

This repository is prepared as a portfolio project for the catfish fry size classification thesis.

## Files

- `Catfish-Fry-Size-Classification_Ronaldo-Firmansyah_ID.pdf` - original Indonesian thesis report in secured PDF format.
- `Catfish-Fry-Size-Classification_Ronaldo-Firmansyah_EN.md` - English portfolio report in Markdown format for GitHub review.
- `Catfish-Fry-Size-Classification_Ronaldo-Firmansyah_EN.pdf` - English secured PDF version for portfolio distribution.

## PDF Security

The secured PDF files were generated with owner-password protection and restricted permissions for copying, extraction, modification, and printing in compliant PDF viewers.

Important limitation: PDF permission restrictions cannot fully prevent screenshots, OCR, or conversion by tools that ignore PDF permissions. The protection is intended to reduce casual copying and modification while keeping the reports viewable for portfolio review.

## Repository Scope

Editable source documents such as DOCX files are intentionally not included. The repository is focused on code, project documentation, and review-ready reports.

## Source Context

The Indonesian thesis report was prepared as a final academic project for Informatics Engineering at State Polytechnic of Malang. The English report was adapted into a concise portfolio version for GitHub and LinkedIn review.
